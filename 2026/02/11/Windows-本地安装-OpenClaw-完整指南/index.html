Windows 本地安装 OpenClaw 完整指南

Posted on 2026-02-11 | In 工具 , AI |
Words count in article: 7.5k | Reading time ≈ 7 mins.

OpenClaw 是一个强大的 AI Agent 平台，支持多种模型和工具集成。本文将详细介绍如何在 Windows 本地安装和配置 OpenClaw，让你能够快速搭建自己的 AI Agent 环境。

前言

OpenClaw 是一个开源的 AI Agent 框架，支持本地部署，可以连接多种大语言模型（如 Moonshot、OpenAI 等），并提供丰富的工具集成（浏览器控制、文件操作、代码执行等）。相比在线服务，本地部署具有更好的隐私性和可控性。

环境准备

在开始安装之前，我们需要准备以下工具：

1. 安装 Git

Git 用于后续可能的源码操作。

下载地址: https://git-scm.com/download/win

安装过程中保持默认选项即可。

2. 安装 nvm-windows

nvm (Node Version Manager) 用于管理 Node.js 版本。

下载地址: https://github.com/coreybutler/nvm-windows/releases

下载 nvm-setup.exe 并安装。安装完成后，打开 PowerShell 验证：

3. 安装 Node.js

⚠️ 重要提示: OpenClaw 要求 Node.js 版本必须 大于等于 22！

使用 nvm 安装 Node.js 22：

为什么不能低于 22 版本？

  • OpenClaw 使用了 Node.js 22 引入的新特性
  • 低版本会导致安装失败或运行时错误
  • 确保使用 nvm use 22 切换到正确版本

总结

通过本文的步骤，你应该已经成功在 Windows 本地安装了 OpenClaw。记住以下关键点：

  1. ✅ Node.js 版本必须 >= 22
  2. ✅ 使用 nvm 管理 Node.js 版本
  3. ✅ npm install 时务必使用代理或镜像
  4. ✅ 使用 openclaw onboard --install-daemon 初始化
  5. ✅ 强烈推荐使用 Moonshot Kimi K2.5 模型
  6. ✅ 妥善保管 API 密钥

现在你可以开始探索 OpenClaw 的强大功能了！
